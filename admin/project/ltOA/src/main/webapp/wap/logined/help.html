帮助中心

1、首页

登录手机WAP，进入首页，页面中间包括通讯录、公告查看、工作流、公文管理、日程安排及事务提醒六个功能模块，下方包括首页、即时消息、通讯录及设置四个菜单。

2、即时消息

进入即时消息列表页面，默认显示最新的即时消息信息，未读的信息由信息数标识；
单击[发即时消息]，进入即时消息信息编辑页面，输入发送内容，单击[发送]，即时消息发送成功;

3、通讯录

通讯录包括个人通讯录及单位通讯录。
进入通讯录列表页面，输入查询条件，列表自动匹配对应的人员信息，单击需要查看的记录，进入人员信息详情页面，页面显示对应人员的基本信息、工作单位信息及家庭信息。人员详情页面可进行发即时消息及拨号操作。

4、公告查看

进入公告查看页面，默认显示置顶和最新的五条公告记录，单击[释放加载更多]列表自动加载其它公告信息，单击对应的公告记录，进入公告详情页面。公告详情页面，可以下载附件。

5、工作流

工作流包括我的申请及审批办理。
进入工作流页面，默认显示我的申请信息及申请记录状态信息，单击需要查看的申请记录，进入申请详情页面，显示申请的流程办理状态、申请内容及附件信息。
单击[审批办理]，显示需要审批的申请记录，单击需要审批的申请记录，进入申批办理页面，输入审批意见，选择下一步处理人，单击[确认转交]，申请审批完成。

6、公文管理

公文管理包括我的发文、收文阅读及公文审批。
我的发文：进入公文管理页面，默认显示我的发文信息及发文状态，单击需要查看的发文，进入发文详情页面，显示流程办理状态、发文信息及附件信息；
收文阅读：单击[收文阅读]，显示收文列表信息，默认未阅读的记录显示在顶端，单击需要查看的收文，进入收文详情页面，显示收文的详细信息及附件信息；
公文审批：单击[公文审批]，进入公文待审批列表，显示需要处理的公文记录，单击需要审批的记录，进入公文审批页面，页面显示公文详情及历史审批记录，公文审批可进行作废、提交及取消操作。

7、电子邮件

电子邮件包括收件箱及发件箱。
进入电子邮件页面，默认显示收件箱记录，未读的邮件显示在顶端，单击需要查看的邮件，进入邮件详情页面，单击[回复]或右上方[写邮件]进入邮件编辑页面，选择收件人员，输入邮件内容，单击[发送]，邮件发送成功。
单击[发件箱]，进入已发邮件列表页面，单击需要查看的邮件，进入邮件详情页面。

8、日程安排

进入日程安排页面，默认显示日程列表信息及信息重要程度，单击需要查看的日程，进入日程详情页面；日程列表页面，单击[新建]进入日程新增页面，选择日程类型、事务类型、优先级、起始时间、结束时间，输入日程信息，单击[保存]，日程新建成功。

9、事务提醒

进入事务提醒页面，默认显示待处理的事务，包括待审批公文、待回复邮件、待处理工作等提醒信息。

10、设置

设置页面包括个人信息、当前OA版本、帮助中心及密码修改；
个人信息：单击[个人信息]，进入人员详细信息页面，包括人员基本信息、工作单位信息及家庭信息； 当前OA版本：显示当前OA版本号；
帮助中心：单击[帮助中心]，进入手机WAP帮助页面；
密码修改：单击[密码修改]，进入密码修改页面，输入正确的新密码，单击[保存]，密码修改成功。
